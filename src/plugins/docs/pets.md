# 宠物系统插件

## 功能概述

宠物系统插件为机器人添加了可领养、喂养和互动的宠物功能，增强用户粘性。

## 命令说明

### 基础命令
- `/宠物` - 显示宠物系统帮助信息
- `/pet` - 英文命令

### 子命令
- `/宠物 领养` - 领养一只宠物
- `/宠物 我的` - 查看自己的宠物
- `/宠物 喂食` - 喂养宠物
- `/宠物 玩耍` - 与宠物互动
- `/宠物 洗澡` - 清洁宠物
- `/宠物 升级` - 升级宠物
- `/宠物 排行` - 查看宠物排行榜

## 宠物类型

### 普通宠物
- **小狗** - 可爱的小狗
- **小猫** - 调皮的小猫
- **兔子** - 温顺的兔子

### 稀有宠物
- **熊猫** - 国宝熊猫
- **老虎** - 威猛的老虎
- **狮子** - 草原之王

### 传说宠物
- **龙** - 神秘的龙
- **凤凰** - 不死的凤凰

## 成长系统

- 宠物有饥饿、快乐、清洁三个属性
- 喂食增加饥饿值
- 玩耍增加快乐值
- 洗澡增加清洁值
- 当三个属性都满时，宠物可以升级

## 升级系统

- 每次升级宠物属性提升
- 升级后宠物外观变化
- 最大等级为10级

## 排行榜

- 按宠物等级排序
- 显示前10名用户
- 每日更新

## 示例

```
/宠物 领养
/宠物 我的
/宠物 喂食
/宠物 玩耍
/宠物 洗澡
/宠物 升级
/宠物 排行
```
