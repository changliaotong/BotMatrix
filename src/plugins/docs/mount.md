# 坐骑系统插件

## 功能概述

坐骑系统插件为机器人添加了可收集、升级和使用的坐骑功能，增强用户互动体验。

## 命令说明

### 基础命令
- `/坐骑` - 显示坐骑系统帮助信息
- `/mount` - 英文命令
- `/ride` - 英文命令

### 子命令
- `/坐骑 商店` - 查看坐骑商店中的所有可用坐骑
- `/坐骑 我的` - 查看自己拥有的坐骑
- `/坐骑 装备 <坐骑ID>` - 装备指定坐骑
- `/坐骑 升级 <坐骑ID>` - 升级指定坐骑
- `/坐骑 排行` - 查看坐骑排行榜

## 坐骑类型

### 陆地坐骑
- **普通战马** - 普通品质，速度+10
- **独角兽** - 稀有品质，速度+30
- **火焰战马** - 史诗品质，速度+50

### 飞行坐骑
- **火焰巨龙** - 传说品质，速度+100
- **不死凤凰** - 神话品质，速度+200

## 升级系统

- 每次升级需要消耗经验值
- 升级后坐骑速度提升10点
- 最大等级为10级

## 排行榜

- 按坐骑总等级排序
- 显示前10名用户
- 每日更新

## 示例

```
/坐骑 商店
/坐骑 我的
/坐骑 装备 1
/坐骑 升级 1
/坐骑 排行
```
