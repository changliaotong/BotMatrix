# Sign In 插件文档

## 概述
签到系统插件支持每日签到和连续签到，提供积分奖励。

## 功能特性

### 手动签到
用户可以通过发送 `!签到` 或 `!sign` 命令进行手动签到。

### 自动签到
用户在群或私聊中发言时，如果今日尚未签到，系统会自动完成签到。

## 命令

### !签到
手动进行签到。

**示例：**
```
!签到
```

### !signstats
查看签到统计信息。

**示例：**
```
!signstats
```

## 积分奖励规则

- 基础奖励：10积分/天
- 连续签到额外奖励：连续签到第N天额外奖励(N-1)积分
- 例如：连续签到第3天，总奖励为10 + 2 = 12积分

## 配置

签到插件不需要额外配置。

## 注意事项

- 每天只能签到一次
- 连续签到天数会在断签后重置为1
- 自动签到会在用户发言时自动触发，不需要额外操作
- 签到成功后会自动发送签到成功消息
