# 机器人功能测试指南

## 如何运行测试

### 方法一：使用批处理脚本
```bash
.	est_user_input.bat
```

### 方法二：直接运行Go程序
```bash
go run cmd/main.go
```

## 测试命令功能

机器人启动后，可以在QQ群或私聊中发送以下命令进行测试：

### 1. 查看所有可用命令
```
/菜单
```

### 2. 群管理命令（需相应权限）
- 设置头衔：`/settitle <用户ID> <头衔>`
- 踢人：`/kick <用户ID> [拒绝入群]`
- 禁言：`/ban <用户ID> [时长]`
- 解除禁言：`/unban <用户ID>`
- 添加管理员：`/addadmin <用户ID>`
- 移除管理员：`/deladmin <用户ID>`

### 3. 其他功能命令
- 查询积分：`/积分 查询`
- 积分排行：`/积分排行`
- 签到：`/签到`
- 天气查询：`/天气 北京`
- 抽签：`/抽签`
- 翻译：`/翻译 Hello World`
- 点歌：`/点歌 稻香`
- 领养宠物：`/领养`
- 查看宠物：`/我的宠物`

## 测试注意事项

1. **权限测试**：
   - `/settitle`：仅群主可用
   - `/kick`、`/ban`、`/unban`：群主或管理员可用
   - `/addadmin`、`/deladmin`：超级管理员可用

2. **参数格式**：
   - 确保命令格式正确，参数之间用空格分隔
   - 用户ID必须是纯数字

3. **测试环境**：
   - 确保数据库和Redis服务正常运行
   - 机器人需要连接到OneBot协议的QQ客户端
