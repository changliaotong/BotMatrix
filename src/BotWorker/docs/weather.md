# Weather 插件文档

## 功能描述
Weather插件是一个天气查询插件，支持查询全球城市的天气信息。

## 命令格式
- `!weather <城市名>`: 查询指定城市的天气信息
- `!天气 <城市名>`: 查询指定城市的天气信息

## 示例
```
用户: !weather 北京
机器人: 当前天气信息
城市: 北京
天气: 晴 (晴朗)
温度: 25.5°C (体感温度: 26.8°C)
最低温度: 20.0°C, 最高温度: 30.0°C
湿度: 65%
气压: 1013 hPa
风速: 3.5 m/s
风向: 180°
云量: 10%
日出: 07:19
日落: 19:19
```

## 配置要求
需要在配置文件中设置OpenWeatherMap API密钥：
```json
{
  "weather": {
    "api_key": "your_api_key_here"
  }
}
```
