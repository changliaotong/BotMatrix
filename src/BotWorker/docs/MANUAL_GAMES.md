# 🎲 趣味互动与工具说明

📜 发送关键词使用指令~

## 🎲 常用游戏
- **猜大小**：`大/小/单/双/围 [积分]`（例：`大 1000`）。
- **三公**：`三公 [积分]`。
- **梭哈**：`梭哈`（全压）。
- **猜数字**：发送 `猜数字 [积分]` 开始，随后发送 `我猜 [数字]`。
- **猜拳**：`剪刀/石头/布 [积分]`。
- **抽奖**：`抽奖 [积分]`。

## 🧩 趣味工具
- **笑话/故事**：发送 `笑话`、`故事` 或 `鬼故事`。
- **成语接龙**：发送 `接龙` 开始，`结束` 终止。
- **成语/反查**：`成语 [词语]` 或 `反查 [意思]`。
- **算命/抽签**：发送 `抽签` 或 `算命`，随后可 `解签`。
- **互动**：`爱群主`、`爱早喵`、`揍群主`、`拍砖`、`逗你玩`。
- **问候**：`早安`、`午安`、`晚安`。
- **彩虹屁**：`彩虹屁`。

## 🛠️ 实用工具
- **翻译**：`翻译 [内容]`。
- **天气**：`天气 [城市]`。
- **拼音**：`拼音 [内容]`。
- **简繁**：`简体 [内容]`、`繁体 [内容]`。
- **计算**：`计算 [算式]`。
- **大写/小写**：`大写 [数字]`、`小写 [数字]`。
- **哈希**：`md5 [内容]`、`sha256 [内容]`。
- **加密/解密**：`加密 [内容]`、`解密 [内容]`。
- **身份证**：`身份证 [号码]`。
- **待办**：`添加待办 [内容]`、`我的待办`、`todo`。
- **时间**：`报时`、`倒计时`。

## 💓 互动系统
- **签到**：`签到`（或 `qd`）。
- **暗恋**：`暗恋 @用户`。
- **变身**：`变身`。
- **头衔**：`我要头衔 [文字]`。
- **宠物**：发送 `宠物` 开启养成之旅。
