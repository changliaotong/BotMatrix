# BotWorker 用户使用手册 (User Manual)

欢迎使用 BotWorker 智能机器人系统！本文档将指导你如何与机器人互动并使用其丰富的功能。

## 🤖 基础交互
机器人支持通过**指令**（如 `!帮助`）和**自然语言**进行交流。你可以直接向机器人提问，它会利用内置的 AI 和知识库为你解答。

## 💰 金融与积分系统
系统采用金融级复式记账法，确保你的每一分资产都安全可靠。
- **签到**：每日签到获取随机积分奖励。
- **积分 / 余额**：查询你当前的积分余额。
- **财务报表**：查看系统整体运行情况和审计记录。

## 🎮 游戏娱乐
BotWorker 提供了多种互动游戏，丰富群聊氛围：
- **成语接龙**：发送 `接龙` 开始游戏。
- **2048**：经典的数字合成游戏。
- **红蓝大战**：群组互动对战游戏。
- **钓鱼**：休闲钓鱼系统，获取稀有鱼类。
- **三公 / 梭哈**：经典的牌类游戏。

## 👨‍👩‍👧‍👦 社交与互动
- **结婚系统**：与其他用户建立虚拟婚姻关系。
- **养娃系统**：共同抚养虚拟后代，体验成长乐趣。
- **宠物系统**：领养并培养你的专属虚拟宠物。
- **运势 / 猜拳**：每日娱乐小功能。

## 🛠️ 系统工具
- **天气 [城市]**：查询指定城市的天气预报。
- **翻译 [内容]**：多语言互译支持。
- **系统信息**：查看机器人运行状态和服务器性能。

## ❓ 常见问题 (FAQ)
**Q: 如何获取更多积分？**
A: 你可以通过每日签到、参与游戏或完成特定任务来获取。

**Q: 机器人不理我怎么办？**
A: 请确保你的消息中包含了正确的触发词，或者确保机器人在该群组中已启用。

---
*提示：本手册已向量化存入机器人大脑。你可以直接问机器人：“如何使用积分系统？”或“有哪些好玩的游戏？”，它会即时为你解答。*
