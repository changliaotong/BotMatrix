# 系统信息插件

## 功能介绍

系统信息插件提供服务器硬件、操作系统、软件版本和性能状态的详细查询功能。

## 命令说明

### 基础命令

| 命令 | 描述 |
|------|------|
| `/系统信息` | 查看完整系统信息 |
| `/systeminfo` | 查看完整系统信息（英文命令） |
| `/sysinfo` | 查看完整系统信息（快捷命令） |

## 查询内容

### 🖥️ 操作系统信息
- 操作系统类型和版本
- 内核版本
- 主机名
- 运行时间

### 🧠 CPU信息
- CPU型号
- 物理核心数
- 逻辑核心数
- CPU占用率

### 💾 内存信息
- 总内存容量
- 已用内存
- 可用内存
- 内存使用率

### 💿 磁盘信息
- 总磁盘容量
- 已用磁盘空间
- 空闲磁盘空间
- 磁盘使用率

### 🐹 Go环境信息
- Go版本
- 编译架构

### 🧵 进程信息
- 当前进程ID
- Goroutine数量

## 使用示例

```
/系统信息
```

## 注意事项

- 系统信息查询需要服务器权限
- 部分信息可能因操作系统不同而有所差异
- 性能数据为实时采集，可能会有波动
- 首次查询可能需要几秒钟时间采集数据
