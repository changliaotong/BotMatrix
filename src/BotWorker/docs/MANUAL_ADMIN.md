# 🛡️ 群管与系统设置

📜 发送关键词使用指令~

## 🛡️ 基础开关
- **开关机**：`开启`（开机）、`关闭`（关机）。
- **全局控制**：`全局开启`、`全局关闭`。
- **功能开关**：`开启 [功能名]`、`关闭 [功能名]`。
  - *可选功能：话痨、阅后即焚、图片回复、撤回回复、语音回复、知识库、积分系统、宠物系统、敏感词、进群确认等。*

## 🛡️ 成员管控
- **警告系统**：
  - `警告 @用户 [原因]`。
  - `查警告 @用户`、`清警告 @用户`。
- **名单管理**：
  - `拉黑 @用户`、`取消拉黑 @用户`、`黑名单`（查列表）。
  - `白名单 @用户`、`取消白名单 @用户`、`白名单`（查列表）。
  - `拉灰 @用户`、`取消拉灰 @用户`。
- **群员处理**：
  - `踢出 @用户`、`禁言 @用户 [时间]`、`禁言我`。
  - `改名 [名字]`、`一键改名`。

## ⚙️ 系统设置
- **权限设置**：
  - `设置管理员 @用户`。
  - `换主人 @用户`。
  - `设置管理权限 [主人/白名单/管理员]`。
  - `设置使用权限 [所有人/白名单/主人]`。
  - `设置调教权限 [所有人/白名单/主人]`。
- **基础设置**：
  - `设置前缀 [符号]`（如：#）。
  - `设置最低积分 [数值]`。
  - `设置城市 [城市]`。
  - `设置系统提示词 [内容]`。

## 🚪 进群与退群
- **欢迎语**：`设置欢迎语 [内容]`、`开启 欢迎语`。
- **退群设置**：`开启 退群提示`、`开启 退群加黑`。
- **改名设置**：`开启 进群改名`、`开启 改名提示`。
- **加群审核**：`设置加群 [忽略/同意/拒绝/密码]`。

## 🖥️ 后台与维护
- **管理后台**：`后台`（获取 Web 管理端链接）。
- **服务状态**：`续费`、`结算`、`换群`。
- **信息查询**：`本群`、`我的权限`。
