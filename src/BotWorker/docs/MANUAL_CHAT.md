# 聊天与 AI 功能手册

本手册介绍机器人的自定义问答（调教）系统以及 AI 智能聊天功能。

## 1. 自定义问答（调教）
您可以教机器人如何回答特定问题。

- **添加问答**：`问 [关键词] 答 [回复内容]`
  - 示例：`问 你好 答 见到你很高兴！`
  - 回复内容支持使用 [系统变量](MANUAL_VARIABLES.md)。
- **删除问答**：`删问 [关键词]`（仅管理员可用）。
- **问答审核**：默认情况下，用户教的内容需要管理员审核后才能全局生效。

## 2. 问答模式设置
管理员可以通过设置不同的模式来控制机器人的说话方式：

- **命令格式**：`设置问答 + [模式]` 或 `开启 [模式]模式`
- **可用模式**：
  - `闭嘴`：不回复任何调教内容，仅响应指令。
  - `本群`：仅回复在本群内教的内容。
  - `官方`：优先回复本群内容，@机器人时回复官方库内容。
  - `话痨`：混合回复本群和官方库内容，回复频率较高。
  - `终极`：混合回复所有群的公开调教内容（需永久版）。
  - `AI`：结合本群/官方知识库，使用 AI 模型生成回复。
  - `纯血AI`：完全脱离调教库，仅使用 AI 模型进行对话。

## 3. AI 智能对话
基于最先进的大语言模型，支持上下文理解。

- **触发方式**：
  - `@机器人 + 问题`
  - `AI + 问题`
  - `聊天 + 问题`
- **系统提示词 (System Prompt)**：
  - 管理员可设置机器人的“人设”：`设置系统提示词 + [内容]`。
  - 示例：`设置系统提示词 你是一个幽默的猫娘。`
- **结束对话**：发送 `结束` 可清除当前 AI 会话的上下文记忆。

## 4. AI 画图
- **命令格式**：`画图 + [描述词]`
  - 示例：`画图 一个戴着草帽的鲁夫在海边吃西瓜。`
- **算力消耗**：生成图片固定消耗 12000 算力/次。

## 5. 算力系统
AI 功能（聊天、画图）需要消耗“算力”。

- **查询算力**：发送 `算力` 查看剩余额度。
- **算力排行**：`算力榜`（本群）或 `算力榜全部`（全服）。
- **获取算力**：
  - 每日签到可获得免费算力。
  - 超级积分可兑换算力。
  - 购买专业版/永久版可获得大量赠送算力。

## 6. 语音与音聊
- **语音回复**：如果开启了 `语音回复` 设置，机器人的部分回答将以语音形式发出。
- **切换音色**：管理员发送 `设置语音` 查看列表，使用 `设置语音 + [编号]` 切换。
