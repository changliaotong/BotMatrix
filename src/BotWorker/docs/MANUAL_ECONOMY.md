# 💰 积分与财富说明

📜 发送关键词使用指令~

## 💎 积分查询
- **我的积分**：直接发送 `积分` 或 `资产`。
- **积分排行榜**：`积分榜`（本群排名）、`积分总榜`（全服排名）。
- **查看排名**：`积分排名`。

## 🏦 银行与存取
- **存入积分**：`存积分 [数值]`（或 `存分`）。
- **取出积分**：`取积分 [数值]`（或 `取分`）。
- **查看存款**：`已存积分`。

## 💸 交易与买卖
- **打赏他人**：`打赏 @用户 [积分]`。
- **转账积分**：`转账 @用户 [积分]`。
- **买入积分**：`买分 [金额]`。
- **卖出积分**：`卖分 [积分]`（将积分兑换为余额）。
- **兑换礼品**：`兑换礼品`。
- **领积分**：`领积分`（免费领取）。

## 💳 余额与算力
- **余额查询**：`余额`（查看账户现金余额）。
- **算力查询**：`算力`（用于 AI 画图等高级功能）。
- **充值续费**：`续费`。

## 💎 金币系统
- **金币查询**：`金币`。
- **金币榜单**：`金币榜`、`金币总榜`。
