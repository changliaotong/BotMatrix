## 2. 待优化的机器人基础功能

### 2.1 消息分发性能
- [ ] **目标**：优化 `PluginManager.DispatchAsync`，减少内存分配，提升大群消息处理能力。

### 2.2 基础指令集 (Base Commands)
- [ ] **帮助指令**：目前仅有简单列表，需实现按分类查看、详细用法说明。
- [ ] **插件管理指令**：实现在线查看插件状态、手动重载特定插件。

### 2.3 富文本支持
- [ ] **目标**：完善对图片、语音、转发消息等特殊消息类型的统一处理封装。

## 3. 开发规范 (针对机器人功能)

- **响应速度优先**：基础机器人功能的响应应尽可能在 200ms 内完成。
- **高可用性**：插件异常不应导致整个 BotWorker 崩溃，必须有完善的 Try-Catch 保护。
- **日志记录**：每一个指令调用必须保留 Audit Log，以便后续 AI 分析用户习惯。
