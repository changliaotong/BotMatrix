# Menu 插件文档

## 功能描述
Menu插件是一个菜单插件，显示所有可用命令。

## 命令格式
- `!菜单`: 显示所有可用命令
- `!menu`: 显示所有可用命令
- `help`: 显示帮助信息
- `!help`: 显示帮助信息

## 示例
```
用户: !菜单
机器人: 🤖 机器人命令菜单
====================

📊 积分系统:
!积分 查询 - 查询当前积分
!积分排行 - 查看积分排行榜

📅 签到系统:
!签到 - 每日签到获取积分

🌤️ 天气查询:
!天气 <城市名> - 查询指定城市天气
!weather <城市名> - 查询指定城市天气

🎲 抽签功能:
!抽签 - 进行一次抽签
!解签 <签文> - 解析签文含义

ℹ️ 其他命令:
!菜单 - 显示此帮助菜单
!help - 显示帮助信息
====================
💡 提示: 所有命令支持中文和英文两种格式
```
