# Pets 插件文档

## 功能描述
Pets插件是一个宠物系统插件，支持领养宠物、喂食、玩耍、洗澡、升级等功能。

## 命令格式
- `!领养`: 领养一只新宠物
- `!领养宠物`: 领养一只新宠物
- `!adopt`: 领养一只新宠物
- `!我的宠物`: 查看所有宠物信息
- `!宠物`: 查看所有宠物信息
- `!pets`: 查看所有宠物信息
- `!喂食 [宠物编号]`: 给宠物喂食
- `!feed [宠物编号]`: 给宠物喂食
- `!玩耍 [宠物编号]`: 和宠物玩耍
- `!play [宠物编号]`: 和宠物玩耍
- `!洗澡 [宠物编号]`: 给宠物洗澡
- `!wash [宠物编号]`: 给宠物洗澡

## 功能特点
- **多种宠物类型**：支持8种不同类型的宠物（猫、狗、兔、仓鼠、熊、熊猫、老虎、狮子）
- **宠物养成**：通过喂食、玩耍、洗澡来照顾宠物
- **等级系统**：宠物可以通过获得经验值升级
- **状态系统**：宠物有饥饿值、快乐值、健康值三种状态属性
- **自动状态变化**：宠物状态会随时间自动变化
- **多宠物支持**：每个用户最多可以领养3只宠物

## 宠物状态说明
- **等级**：宠物的等级，通过积累经验值升级
- **经验**：宠物的经验值，达到当前等级×100时自动升级
- **饥饿值**：0-100，过低会影响健康值
- **快乐值**：0-100，过低会影响健康值
- **健康值**：0-100，反映宠物的健康状况

## 状态变化规则
- **自动变化**：每小时减少5点饥饿值和5点快乐值
- **健康影响**：饥饿值或快乐值低于20时，每小时减少10点健康值
- **升级效果**：升级后健康值和快乐值恢复满，饥饿值恢复到80
