# BotWorker - OneBot协议兼容机器人处理程序 (C# 重构版)

BotWorker 是一个使用 C# ( .NET 10) 编写的兼容 OneBot 协议的机器人处理程序。它通过吸收 sz84 的核心逻辑并进行现代化重构，提供了高性能、类型安全且易于扩展的插件系统。

## 功能特性

- ✅ **现代技术栈**: 基于 .NET 10 构建，利用最新的 C# 特性（如 Nullable Reference Types, Primary Constructors 等）。
- ✅ **OneBot v11 兼容**: 支持 WebSocket 和 HTTP 通信方式。
- ✅ **模块化插件系统**: 提供灵活的插件加载机制。
- ✅ **跨平台支持**: 核心逻辑解耦，支持 Windows 特有功能优化。
- ✅ **高性能**: 采用异步处理机制，优化并发消息处理。

## 核心重构说明

1. **语言迁移**: 从 Go 迁移至 C#，统一了团队技术栈。
2. **逻辑内化**: 完整吸收了 `sz84` 核心库的代码，实现了自包含架构，不再依赖外部 `sz84.common`。
3. **性能优化**: 优化了消息格式化（FriendlyMessage）和缓存机制（Redis 缓存集成）。
4. **警告清理**: 解决了所有 CS8603 (Null 引用) 和 CA1416 (平台兼容性) 警告。

## 🚀 快速开始

1. 确保安装了 .NET 10 SDK。
2. 配置 `appsettings.json` 中的数据库和 Redis 连接。
3. 运行项目：
   ```bash
   dotnet run --project src/BotWorker/BotWorker.csproj
   ```
