This catalog features some of the advanced built-in plugins and specialized systems available in the BotMatrix ecosystem.

---

## 1. Fission System (Growth Engine)
The Fission System encourages viral growth through incentive mechanisms.
- **Invitation Codes**: Users generate codes to invite friends.
- **Rewards**: Points and badges for successful invitations and task completion.
- **Anti-Fraud**: Built-in validation to prevent self-binding and fake account abuse.

## 2. Baby & Marriage System (Social Interaction)
Enhance community engagement with virtual social systems.
- **Marriage**: Propose, marry, and interact with partners. Includes wedding sweets and "sweet hearts."
- **Baby**: Adopt and nurture virtual babies with a growth and leveling system.

## 3. QQ Guild Smart Collaboration
Specialized logic for official QQ Guild Bots to overcome platform restrictions.
- **Smart Send**: Automatically uses auxiliary regular bots to "wake up" the Guild Bot.
- **Credential Caching**: Manages message IDs and validity windows for seamless replies.

## 4. Super Group Manager
An advanced moderation plugin built with the C# SDK.
- **Smart Interception**: Real-time scanning for sensitive content and auto-muting.
- **Interactive Config**: Administrators can configure settings via an in-chat interactive menu (`!config`).

## 5. Standard Utilities
- **Weather & Translation**: Real-time queries and multi-language translation.
- **Points & Sign-in**: Gamified engagement with daily check-ins and a points-based economy.
- **Moderation**: Comprehensive tools for muting, kicking, and blacklisting.
