# 麦克风配置与授权指南

本指南详细说明了如何在 Windows 10 系统中授权应用使用麦克风，以及如何在远程桌面连接中配置并使用麦克风。

## 1. Windows 10 应用麦克风授权

在 Windows 10 中授权应用使用麦克风，请按照以下步骤操作：

### 1.1 开启系统级麦克风访问权限
- 点击任务栏左侧的 **“开始”** 菜单，选择 **“设置”**（齿轮图标）。
- 在设置界面中选择 **“隐私”**。
- 在左侧菜单栏中点击 **“麦克风”**。
- 确保顶部显示 **“此设备的麦克风访问权限已打开”**。若显示关闭，请点击 **“更改”** 按钮并切换为 **“开”**。

### 1.2 授权 Microsoft Store 应用
- 在同一页面，向下滚动到 **“允许应用访问你的麦克风”**，确保开关处于 **“开”** 状态。
- 在 **“选择哪些 Microsoft Store 应用可以访问你的麦克风”** 列表中，找到目标应用并打开开关。

### 1.3 授权桌面应用（传统软件）
- 滚动到页面底部，找到 **“允许桌面应用访问你的麦克风”**。
- 确保此开关处于 **“开”** 状态。这控制着如 Chrome、微信、QQ 等非应用商店软件的访问权限。

## 2. 远程桌面（RDP）使用麦克风

在远程桌面连接中使用本地麦克风，需要同时配置客户端和服务器端。

### 2.1 本地客户端配置（连接前）
- 启动 **远程桌面连接** (`mstsc`)。
- 点击 **“显示选项”** -> **“本地资源”** 选项卡。
- 在“远程音频”部分点击 **“设置...”**。
- 在“远程音频录制”下选择 **“在此计算机上录制”**，然后点击确定。

### 2.2 远程服务器端配置（组策略）
若远程电脑无法识别麦克风，需在**远程电脑**上开启策略：
- 按下 `Win + R`，输入 `gpedit.msc` 并回车。
- 导航至：`计算机配置` -> `管理模板` -> `Windows 组件` -> `远程桌面服务` -> `远程桌面会话主机` -> `设备和资源重定向`。
- 双击 **“允许音频录制重定向”** 并设置为 **“已启用”**。
- 重启远程电脑或运行 `gpupdate /force`。

### 2.3 权限验证
- 在远程电脑的 **设置** -> **隐私** -> **麦克风** 中确保已开启访问权限。
- 在声音设置的“输入”设备中，应能看到名为 **“远程音频”** 的设备。

## 常见问题排查
- **浏览器权限**：若在网页中使用，需点击浏览器地址栏左侧的“锁头”图标授予权限。
- **驱动检查**：若设备不可用，请通过“设备管理器”检查音频驱动是否异常。
- **系统服务**：远程服务器需确保 `Windows Audio` 服务处于运行状态。
