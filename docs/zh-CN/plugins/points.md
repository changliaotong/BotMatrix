# Points 插件文档

## 功能描述
Points插件是一个积分系统插件，支持签到积分、发言积分、查询积分和积分排行榜功能。

## 命令格式
- `!points`: 查询当前积分
- `!积分`: 查询当前积分
- `!signpoints`: 签到获得积分
- `!签到积分`: 签到获得积分
- `!rank`: 查看积分排行榜
- `!排行榜`: 查看积分排行榜
- `!积分榜`: 查看积分排行榜

## 示例
```
用户: !积分
机器人: 你当前的积分为：100
```

```
用户: !签到积分
机器人: 签到成功！获得10积分
当前积分：110
```

```
用户: !排行榜
机器人: 🏆 积分排行榜 🏆
------------------------
🥇 用户123456：200积分
🥈 用户789012：150积分
🥉 用户345678：100积分
4. 用户901234：80积分
5. 用户567890：70积分
------------------------
总参与人数：100人
```

## 功能特点
- **签到积分**：每日签到获得10积分
- **发言积分**：每发送一条非命令消息获得1积分
- **积分查询**：随时查看当前积分
- **积分排行榜**：查看前10名的积分排名
- **积分记录**：记录所有积分变动情况
