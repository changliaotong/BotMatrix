# Utils Plugin Documentation

## Overview
The Utils plugin provides various utility features for the bot, including:
- 报时 (Time)
- 计算 (Calculation)
- 鬼故事 (Ghost Stories)
- 成语接龙 (Idiom Solitaire)
- 笑话 (Jokes)

## Commands

### !报时
Get the current time.

**Examples:**
```
!报时
```

### !计算 <expression>
Perform a simple calculation.

**Parameters:**
- `<expression>`: The mathematical expression to calculate (supports +, -, *, /)

**Examples:**
```
!计算 1+2
!计算 3*4
!计算 5-6
!计算 7/8
```

### !鬼故事
Get a random ghost story.

**Examples:**
```
!鬼故事
```

### !成语接龙
Start an idiom solitaire game.

**Examples:**
```
!成语接龙
```

### !笑话
Get a random joke.

**Examples:**
```
!笑话
```

## Configuration

The Utils plugin does not require any additional configuration.

## Notes
- Calculations are limited to basic arithmetic operations
- Ghost stories and jokes are randomly selected from a pre-defined list
- Idiom solitaire game requires user participation to continue
