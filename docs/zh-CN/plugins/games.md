# Games Plugin Documentation

## Overview
The Games plugin provides various interactive games for the bot, including:
- 猜拳 (Rock-Paper-Scissors)
- 猜大小 (Big or Small)
- 抽奖 (Lottery)
- 三公 (Three Cards)
- 梭哈 (Showdown)

## Commands

### !猜拳 <choice>
Play Rock-Paper-Scissors with the bot.

**Parameters:**
- `<choice>`: Can be "石头" (rock), "剪刀" (scissors), or "布" (paper)

**Examples:**
```
!猜拳 石头
!猜拳 剪刀
!猜拳 布
```

### !猜大小 <choice>
Play Big or Small game with the bot.

**Parameters:**
- `<choice>`: Can be "大" (big) or "小" (small)

**Examples:**
```
!猜大小 大
!猜大小 小
```

### !抽奖
Participate in a lottery draw.

**Examples:**
```
!抽奖
```

### !三公
Play Three Cards game.

**Examples:**
```
!三公
```

### !梭哈
Play Showdown game.

**Examples:**
```
!梭哈
```

## Configuration

The Games plugin does not require any additional configuration.

## Notes
- All games are for entertainment purposes only
- No real money or points are involved in the games
- The bot uses random number generation to determine game outcomes
- Game results are final and cannot be changed
