# 菜单插件

## 功能概述

菜单插件提供机器人所有命令的汇总展示，帮助用户快速了解可用功能。

## 命令说明

### 基础命令
- `/菜单` - 显示完整命令菜单
- `/menu` - 英文命令
- `/help` - 英文命令

## 菜单结构

### 积分系统
- `/积分 查询` - 查询个人积分
- `/积分排行` - 查看积分排行榜

### 签到系统
- `/签到` - 每日签到领积分

### 天气查询
- `/天气` - 查询实时天气
- `/weather` - 英文命令

### 抽签功能
- `/抽签` - 抽取运势签
- `/解签` - 解释签文

### 翻译功能
- `/翻译` - 中英文互译
- `/translate` - 英文命令

### 点歌功能
- `/点歌` - 搜索并播放歌曲
- `/music` - 英文命令

### 宠物系统
- `/领养` - 领养宠物
- `/我的宠物` - 查看自己的宠物
- `/喂食` - 喂养宠物
- `/玩耍` - 与宠物互动
- `/洗澡` - 清洁宠物

### 送礼物系统
- `/送礼物` - 给其他用户送礼物
- `/礼物列表` - 查看礼物列表
- `/抽礼物` - 抽取礼物
- `/drawgift` - 英文命令

### 爱群主系统
- `/爱群主` - 发送爱群主消息
- `/loveowner` - 英文命令
- `/粉丝团排行` - 查看粉丝团排行榜
- `/fanrank` - 英文命令

### 管理员命令
- `!kick` - 踢出群成员
- `!ban` - 禁言群成员
- `!unban` - 解除禁言
- `!addadmin` - 添加管理员
- `!deladmin` - 删除管理员
- `!setrules` - 设置群规则
- `!addword` - 添加敏感词
- `!delword` - 删除敏感词
- `!members` - 查看群成员列表
- `!memberinfo` - 查看成员信息
- `!settitle` - 设置成员头衔
- `!invitationstats` - 查看邀请统计
- `!inviterank` - 查看邀请排行榜

### 成就系统
- `/成就` - 查看成就系统
- `/achievement` - 英文命令
- `/achieve` - 英文命令

### 坐骑系统
- `/坐骑` - 查看坐骑系统
- `/mount` - 英文命令
- `/ride` - 英文命令

### 其他命令
- `/时间` - 查看当前时间
- `/time` - 英文命令
- `/问答` - 问答系统
- `/qa` - 英文命令
- `/系统信息` - 查看服务器信息
- `/systeminfo` - 英文命令
- `/sysinfo` - 英文命令

## 提示

发送任意命令即可查看该命令的详细使用说明。
