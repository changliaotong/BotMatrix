# 超级多级菜单系统 (Menu System)

## 1. 核心特性
- **无限层级**：支持嵌套容器（Container），理论上支持无限深度的菜单结构。
- **多轮对话 (Multi-turn Conversation)**：支持 `Input` 类型的节点，可以连续向用户提问并采集数据。
- **状态持久化**：基于 `MenuSession` 维护每个用户的交互位置、已采集数据和活跃时间。
- **视觉增强**：自动渲染为美观的 ASCII 边框，适配各类聊天终端。

## 2. 节点类型说明
- `Container`: 导航节点，点击后进入下一级子菜单。
- `Command`: 触发特定的 `Skill`（如开启坐骑系统、查询余额）。
- `Input`: 启动一个采集流程，按照预设问题列表依次询问用户。
- `Back`: 返回上一级菜单。

## 3. 指令说明
- `菜单` / `menu`: 进入主控中心。
- `数字 (1, 2, ...)`: 在菜单模式下，直接输入数字进行导航。
- `退出菜单`: 销毁当前会话并返回普通聊天模式。

## 4. 扩展开发
在 `MenuService.cs` 的 `InitializeMenuTree` 方法中，您可以动态构建复杂的业务逻辑。未来可以扩展为从 JSON 配置文件加载菜单树。
