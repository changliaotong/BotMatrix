# Music 插件文档

## 功能描述
Music插件是一个点歌插件，支持搜索歌曲并播放。

## 命令格式
- `!点歌 <歌曲名称>`: 搜索并播放指定歌曲
- `!music <歌曲名称>`: 搜索并播放指定歌曲

## 示例
```
用户: !点歌 晴天
机器人: 🎵 正在为您点歌：晴天
请点击链接播放：https://music.163.com/#/search/m=晴天
```

```
用户: !music Hello
机器人: 🎵 正在为您点歌：Hello
请点击链接播放：https://music.163.com/#/search/m=Hello
```
