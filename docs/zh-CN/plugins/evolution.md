# 🆙 Evolution 进化系统插件

`EvolutionService` 是 BotMatrix 的核心激励插件，负责处理用户等级、经验、勋章以及荣耀榜单。

## 1. 核心功能

- **经验自动增长**：监听 `PointTransactionEvent`，将积分变动按比例（1:1）转化为经验值。
- **等级系统**：根据经验值自动计算等级，采用二次方增长公式（`Level = sqrt(exp/100) + 1`）。
- **勋章系统**：基于用户行为和属性自动授予勋章。
- **荣耀榜单**：展示全服等级排名前 10 的用户。
- **新手任务**：监听 `SystemInteractionEvent`，为首次互动的用户发放 50 经验及勋章。

## 2. 交互指令

在机器人对话框或超级菜单中输入以下指令：

| 指令 | 描述 |
| :--- | :--- |
| `evolution` | 查看自己的当前等级、经验、升级进度及已获勋章 |

## 3. 勋章列表

| 勋章名称 | 获得条件 |
| :--- | :--- |
| 💰 第一桶金 | 经验值大于 0 |
| 🏅 崭露头角 | 等级达到 5 级 |
| 🔥 矩阵精英 | 等级达到 10 级 |
| 👑 进化主宰 | 等级达到 50 级 |
| 💎 积分大亨 | 经验值达到 10,000 |
| ⛵ 新手启航 | 首次打开超级菜单或与系统互动 |

## 4. 开发者集成

### 订阅等级变动
如果您想在用户升级时给予奖励，可以订阅 `LevelUpEvent` (暂定)。

### 自定义经验来源
您可以发布 `PointTransactionEvent` 来间接增加用户经验。
