# SuperGroupManager 插件文档

[English](./super_group_manager_en.md) | 简体中文

## 🌟 插件简介
`SuperGroupManager` 是一个基于 BotMatrix C# SDK (v3.0) 开发的高级群组管理插件。它不仅提供基础的禁言、踢人功能，还引入了智能中间件拦截系统和异步交互式配置面板，是群组自动化管理的强力工具。

## ✨ 核心功能
- **智能中间件拦截**: 实时扫描群消息，自动识别并拦截敏感词，执行撤回与禁言惩罚。
- **入群欢迎系统**: 自动识别新成员入群，发送个性化欢迎消息。
- **交互式配置面板**: 管理员可通过 `!config` 指令唤起交互菜单，通过对话式交互修改插件设置。
- **权限管理**: 严格遵循 BotMatrix 权限体系，确保只有授权管理员可执行敏感操作。

## 🛠️ 安装与使用

### 安装
1. 下载 `SuperGroupManager.bmpk` 插件包。
2. 通过 BotMatrix 控制面板或 `bm-cli` 加载插件：
   ```bash
   ./bm-cli load ./SuperGroupManager.bmpk
   ```

### 指令说明
| 指令 | 描述 | 权限要求 |
| :--- | :--- | :--- |
| `!config` | 开启交互式配置面板 | 群管理员/插件管理员 |

### 交互配置选项
唤起 `!config` 后，您可以根据提示输入数字选择：
1. **开启/关闭 入群欢迎**
2. **编辑 敏感词库**
3. **设置 自动禁言时长**
4. **退出设置**

## 🛡️ 安全性说明
本插件需要以下 BotMatrix 权限声明：
- `mute_user`: 用于自动禁言违规用户。
- `delete_message`: 用于撤回敏感消息。
- `send_message`: 用于发送欢迎语和配置反馈。
