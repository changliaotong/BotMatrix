# 📖 BotMatrix 用户全景指南 (User Guide)

欢迎使用 BotMatrix！本指南将带您从零开始了解“数字员工”与“技能机器人”生态，涵盖从快速上手到高级 AI 能力、群组治理及服务端运维的全方位内容。

---

## 🌟 核心概念：什么是 BotMatrix？

BotMatrix 是一套 **Agent OS（智能体操作系统）**。它将传统的聊天机器人升级为具备企业属性的虚拟资产。

- **数字员工 (Digital Employee)**：拥有姓名、职位、性格和 KPI 的 AI 形象（如：您的财务助理“小王”）。
- **技能机器人 (Skill Bot)**：赋予数字员工的具体能力。一个员工可以挂载“查天气”、“翻译”、“写代码”等多个技能机器人。
- **技能 (Skills/Tools)**：数字员工能执行的具体任务，通过 **MCP 协议** 或原生插件实现。

---

## 🚀 快速上手：三步雇佣你的数字员工

### 第一步：创建数字员工
在 Overmind 管理后台创建一个新的数字员工档案。
- **设定性格**：严谨的工程师、幽默的伴侣或专业的客服。
- **配置人设**：定义其工作职责和知识边界。

### 第二步：配置技能集
为数字员工挂载所需技能：
- **基础技能**：聊天、提醒、群管。
- **高级技能**：接入 MCP 工具集（如 GitHub 审查、数据库查询）。
- **知识库**：通过 RAG 上传业务文档，让它成为行业专家。

### 第三步：派往工作岗位
将数字员工连接到 QQ、微信、飞书或 Discord。现在，它就可以正式开始为您和您的团队服务了！

## 🧠 高级 AI 能力 (Advanced Features)

### 1. RAG 2.0 知识库 (Nexus RAG)
不仅仅是简单的搜索，而是具备**自省与推理**能力的第二大脑。
- **全格式支持**：直接在聊天窗口发送 PDF、Word、Excel、代码或图片，机器人会自动索引。
- **场景隔离**：私聊文档仅个人可见，群聊文档全群共享。
- **RAG 2.0 特性**：支持查询改写、自我反思（不知道就不瞎编）和跨文档知识漫游。

### 2. MCP 统一能力接口
采用 Anthropic 标准协议，实现工具的“即插即用”。
- **能力平权**：任何符合 MCP 标准的工具（如搜索、天气、计算）都能秒级接入。
- **自主调度**：数字员工能根据您的意图，自动组合多个 MCP 工具完成复杂任务。

### 3. 全球智能体网格 (Global Agent Mesh)
**打破企业边界的协作**：您的数字员工可以安全地调用合作伙伴公司的专业 AI（如：您的秘书向物流公司的 AI 查询快递进度）。所有通信均经过 JWT 认证和隐私脱敏。

## 🛡️ 群组治理与自动化 (Group Management)

通过 **超级群管 (Super Group Manager)** 插件实现高度自动化的群运营：

- **违规监控**：实时扫描关键词、正则表达式。支持阶梯式禁言（5分钟 -> 1小时 -> 24小时）。
- **经济杠杆**：违规自动扣除积分，邀请新成员自动奖励积分。
- **智能欢迎**：新成员入群自动艾特、发送群规、引导修改名片。
- **免指令配置**：管理员发送 `/config` 即可通过交互式菜单配置群规则。

## ⚙️ 管理员权限说明 (Admin Setup)

系统提供三层权限模型：
- **RobotOwner (机器人主人)**：管理自己接入的自有机器人及对应群组。
- **GroupOwner (群主)**：仅配置所在群组的插件开关与参数。
- **Admin (系统管理员)**：拥有全局配置权限，管理所有机器人和系统参数。

**机器人类型差异**：
- **自有机器人**：用户自行接入，拥有完整协议配置权限。
- **官方机器人**：系统统一提供，用户仅能配置群组层面的插件功能。

## �️ 服务端运维与指令 (Server Manual)

管理员可通过 Web 控制台或聊天指令管理系统。

### 1. Web 控制台
- **地址**：默认 `http://服务器IP:5000` (管理后台) 或 `http://IP:3001` (网关监控)。
- **功能**：查看 CPU/内存看板、实时日志、二维码登录、消息吞吐量。

### 2. 管理员 `#` 指令
仅限管理员在聊天窗口使用：
- `#status`：查看系统负载、运行时间、连接数。
- `#reload`：热重载所有插件配置（无需重启）。
- `#broadcast <消息>`：向所有活跃群组发布系统公告。
- `#db_clean <天数>`：清理过期的聊天记录，释放数据库空间。

## ❓ 常见问题 (FAQ)

- **Q: 使用 BotMatrix 需要付费吗？**
  - A: 核心框架开源免费。您仅需支付给 AI 服务商（如 OpenAI、DeepSeek）产生的 Token 费用。
- **Q: 如何保护我的隐私？**
  - A: 系统内置“隐私堡垒”，外发给 AI 的消息会自动脱敏手机号、身份证等敏感信息，并在返回时还原。
- **Q: 为什么机器人不理我？**
  - A: 请检查 `#status` 是否正常，或在 Web 控制台确认机器人是否处于登录状态。
