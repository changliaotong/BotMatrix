## 🌟 什么是 BotMatrix？

你可以把 BotMatrix 想象成一家**虚拟猎头公司**。
在这里，你可以“雇佣”具备各种能力的**数字员工**（机器人），并为它们配置不同的**技能**（插件），让它们在 QQ、微信、飞书等平台为你工作。

### 核心概念：
1. **数字员工 (Digital Employee)**：一个有名字、有职位、有性格的 AI 形象（比如：你的财务助理“小王”）。
2. **技能机器人 (Skill Bot)**：赋予数字员工的具体能力。一个数字员工可以同时拥有“查天气”、“翻译”、“写代码”等多个技能机器人属性。
3. **技能 (Skills/Tools)**：数字员工能执行的具体任务。

---

## 🚀 快速上手：三步雇佣你的数字员工

### 第一步：挑选你的数字员工
在管理后台，你可以创建一个新的数字员工档案。
- **起个好听的名字**：比如“研发助理·小林”。
- **设定性格**：你可以让它表现得像个严谨的工程师，或者像个幽默的伙伴。

### 第二步：为它配置技能机器人能力
没有技能的数字员工只是个“聊天机器”。你需要为它挂载技能：
- **基础技能**：内置的聊天、提醒等功能。
- **高级技能**：通过 **MCP 接口** 接入的专业能力，比如“GitHub 代码审查”、“数据库查询”。

### 第三步：派往工作岗位
将你的数字员工连接到你常用的平台（如：加入你的工作群）。
现在，它就可以正式开始工作了！

---

## 🌈 为什么选择 BotMatrix？

- **它有记忆**：你的数字员工会记得你交代过的事，不会像普通机器人那样“转头就忘”。
- **它有朋友圈**：通过 **Global Agent Mesh**，你的数字员工可以向其他公司的专业员工请教问题（跨企业协作）。
- **它很安全**：所有的私密信息都会被“隐私堡垒”过滤，不用担心泄露给外部 AI 模型。

## 💰 费用说明

- **软件免费**：BotMatrix 是一个开源项目，您可以自由地将其部署在自己的服务器上，无需支付软件授权费。
- **服务收费**：
  - **算力成本**：AI 回复消耗的 Token 费用由您直接支付给 AI 服务商（如 OpenAI、DeepSeek 等）。
  - **增值服务**：如果您需要官方提供的专家部署、定制化技能机器人开发或大规模网格管理服务，我们将收取相应的服务费用。
