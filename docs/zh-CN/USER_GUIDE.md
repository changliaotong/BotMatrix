# BotMatrix 用户入门指南：打造你的全能数字助手 🤖

欢迎使用 BotMatrix！如果你觉得“Global Agent Mesh”、“MCP”这些词听起来像天书，别担心，这篇文章就是为你准备的。我们将用最简单的话告诉你：**BotMatrix 能帮你做什么，以及如何开始。**

---

## 🌟 1. BotMatrix 是什么？

简单来说，BotMatrix 是一个**超级管家系统**。它能把你常用的聊天软件（如微信、钉钉、Discord）变成一个聪明的“超级助手”。

你可以把它想象成：
- **耳朵和嘴巴**：它能连接你的微信、QQ 或钉钉，听你说话，并回复你。
- **大脑**：它连接了最先进的 AI（如 ChatGPT、Claude、豆包等），能理解你的意图。
- **双手**：它能通过各种“技能包”，帮你查天气、订会议室、甚至管理你的个人日程。

---

## 💡 2. 我能用它做什么？

### 🏠 个人场景
- **全能助理**：在微信里发一句“帮我总结一下明天的日程”，它就会自动查好告诉你。
- **知识库管家**：把你的学习资料发给它，以后随问随答。
- **智能提醒**：让它在特定时间或特定条件下（比如“如果明天下雨”）提醒你。

### 🏢 办公场景
- **数字员工**：帮你在群里自动回复常见问题，接待新客户。
- **自动化办公**：发个指令，它就能帮你把表格数据整理好，或者发一封邮件。
- **团队协作**：多个机器人之间可以“交头接耳”，共同完成复杂的项目任务。

## 🛠️ 3. 核心概念（白话版）

为了让你更好地使用，这里有三个你可能会看到的词，我们用生活中的例子来解释：

1.  **适配器 (Adapters) = “翻译官”**
    - 它的作用是让 BotMatrix 学会说不同平台的“语言”。有了它，你的助手就能同时在微信、QQ 和飞书上上班。
2.  **MCP 技能包 = “助手的工具箱”**
    - 以前机器人只会聊天，现在通过 MCP，你可以给它安装“查快递工具”、“搜论文工具”或“画图工具”。一次安装，所有平台的机器人都能用。
3.  **智能体网格 (Mesh) = “助手的社交圈”**
    - 你的助手不再是孤军奋战。如果你的助手不会某个技能，它可以去请教（调用）其他公司或朋友公开的“专家助手”，安全地帮你把事办成。

---

## 🚀 4. 如何开始使用？

只需三步，开启你的 AI 助手之旅：

1.  **第一步：启动系统**
    - 按照 [部署指南](core/DEPLOY.md) 运行程序，你会看到一个管理后台。
2.  **第二步：连接你的账号**
    - 在后台选择你想用的平台（比如微信），扫码或输入配置，让助手的“耳朵和嘴巴”上线。
3.  **第三步：挑选技能**
    - 在“技能中心”开启你感兴趣的功能。现在，你就可以在聊天窗口调戏你的 AI 助手啦！

---

## 🛡️ 5. 我的隐私安全吗？

**非常安全！** 
BotMatrix 像一个保险柜。你的所有聊天记录和私密数据都会经过自动过滤。当助手去“请教”其他专家时，它会自动把你的名字、电话等隐私遮住，只把问题发出去。

## ❓ 常见问题

- **Q: 使用这个需要付钱吗？**
  - A: BotMatrix 本身是开源免费的。但如果你连接了某些 AI 大脑（如 OpenAI），可能需要向它们支付 API 费用。
- **Q: 我需要会写代码吗？**
  - A: 如果只是使用现成功能，完全不需要！如果你想自己开发新技能，我们准备了 [开发手册](plugins/PLUGIN_DEVELOPMENT.md)。
