# 技能中心与训练场 (Skill Center & Training Center)

> [⬅️ 返回文档中心](README.md) | [🏠 返回项目主页](../../README.md)

BotMatrix 并非简单的消息转发工具，而是一个 **AI 技能孵化平台**。通过“技能中心”与“技能训练中心”，用户可以零代码地赋予机器人强大的认知与执行能力。

## 1. 技能中心 (Skill Center)

技能中心是机器人能力的“应用商店”，负责技能的发现、分发与挂载。

### 核心功能
- **技能货架**: 展示系统预置技能（如：智能翻译、代码评审）与用户自定义技能。
- **一键挂载**: 用户可以将技能直接绑定到特定的机器人实例或群组。
- **热插拔设计**: 技能的开启与关闭即时生效，无需重启任何服务。
- **隔离机制**: 租户间的技能完全隔离，支持私有技能、组织内共享技能与全局公开技能。

## 2. 技能训练中心 (Skill Training Center)

训练中心是技能的“进化实验室”，通过数据驱动让 AI 表现持续提升。

### 核心功能
- **提示词 (Prompt) IDE**: 
    - 可视化编辑 System Prompt 与 User Prompt。
    - 支持变量动态注入（如：`{{.UserName}}`, `{{.LastMessage}}`）。
    - 多版本对比预览。
- **语料标注与纠错**:
    - 自动抓取 AI 回复效果不佳的对话。
    - 提供手动修正界面，将修正后的数据转化为 Few-shot 示例。
- **RAG 实验室**:
    - 在线上传知识库文档 (PDF, Markdown, Docx)。
    - 调试分段 (Chunking) 策略与检索准确度。
- **自动化微调 (Fine-tuning)**:
    - 集成主流提供商的微调接口。
    - 当标注数据达到阈值时，支持一键启动模型定制化训练。

## 3. 技能开发流程

1. **定义场景**: 在技能中心创建新技能，设定应用场景（如：售后客服）。
2. **编写提示词**: 在训练中心设计初始 Prompt。
3. **挂载测试**: 将技能挂载到测试机器人。
4. **持续调优**: 根据实际对话反馈，在训练中心进行标注与迭代。

## 4. 数据架构支持

系统底层通过 `AISkillGORM` 与 `AITrainingDataGORM` 等模型支撑技能的持久化与训练数据的流转，确保每一条训练数据都能闭环反馈到模型侧。
