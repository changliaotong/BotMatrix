# 宝宝系统插件

## 功能概述

宝宝系统插件为机器人添加了完整的宝宝养成功能，包括宝宝降临、成长值系统、生日系统、互动功能、商城系统、打工功能、改名功能等，增强用户与机器人的互动体验。宝宝系统支持成长值累积、年龄计算、年度生日系统等核心机制。

## 命令说明

### 普通用户命令
- **宝宝降临** - 获得一个新宝宝
- **我的宝宝** - 查询当前宝宝的详细信息
- **宝宝学习** - 增加宝宝的成长值
- **宝宝互动** - 与宝宝互动增进感情
- **宝宝商城** - 查看宝宝用品商城
- **购买+商品编号** - 购买指定商品（如：购买1）
- **宝宝打工** - 宝宝打工获得成长值和积分
- **宝宝改名+新名字** - 给宝宝改名字（如：宝宝改名+小宝贝）
- **拐卖宝宝说明** - 查看宝宝系统的相关说明和使用规则

### 超级管理员命令
- **开启宝宝系统** - 开启整个宝宝系统
- **关闭宝宝系统** - 关闭整个宝宝系统
- **抛弃宝宝QQ** - 处理不当使用宝宝系统的用户（超管专用）

## 成长值系统

- 每获得1000点成长值，宝宝年龄增加1天（成长值每1000点+1天）
- 每30天年龄增加1级
- 每日自动增加50点成长值
- 宝宝学习可获得100点成长值
- 宝宝互动可获得50点成长值
- 宝宝打工可获得150点成长值
- 购买商城商品可获得相应成长值加成

## 生日系统

- 宝宝每年过一次生日（生日每1岁过一次）
- 生日日期与宝宝降临日期相同
- 生日当天会收到系统通知并获得特殊奖励

## 商城系统

### 商品列表
1. **奶瓶** - 50积分，增加100成长值
2. **玩具车** - 100积分，增加200成长值
3. **故事书** - 150积分，增加300成长值
4. **新衣服** - 200积分，增加400成长值

### 购买方式
发送 `购买+商品编号` 即可购买对应商品，如：`购买1`

## 互动系统

- **宝宝学习** - 提升宝宝知识水平
- **宝宝互动** - 增进亲子感情
- **宝宝打工** - 培养宝宝独立性（需30天以上年龄）

## 打工系统

- 宝宝需达到30天（1个月）以上才能打工
- 每次打工获得150点成长值和50积分
- 打工可以培养宝宝的责任感和独立性

## 超管功能

- 超级管理员可开启/关闭宝宝系统
- 超级管理员可处理不当使用宝宝系统的用户
- 超级管理员可执行系统维护操作
