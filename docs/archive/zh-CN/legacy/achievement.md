# 成就系统插件

## 功能介绍

成就系统插件为机器人提供了一套完整的成就管理系统，允许用户通过完成各种任务和活动来解锁成就并获得奖励。

## 命令说明

### 基础命令

| 命令 | 描述 |
|------|------|
| `/成就` | 显示成就系统帮助信息 |
| `/achievement` | 显示成就系统帮助信息（英文命令） |
| `/achieve` | 显示成就系统帮助信息（快捷命令） |

### 子命令

| 命令 | 描述 |
|------|------|
| `/成就 列表` | 查看所有可用成就 |
| `/成就 我的` | 查看已获得的成就 |
| `/成就 进度` | 查看成就完成进度 |
| `/成就 排行` | 查看成就排行榜 |

## 成就类型

### 🎓 新手成就
- **首次签到** - 完成第一次签到
- **连续签到** - 连续签到7天

### 👥 社交成就
- **送礼达人** - 累计送出100份礼物
- **群主真爱粉** - 累计发送100次爱群主

### 🎯 活跃成就
- **话唠之王** - 累计发送1000条消息

## 成就奖励

每个成就都有对应的积分奖励：

| 成就 | 奖励积分 |
|------|----------|
| 首次签到 | 100 |
| 连续签到 | 500 |
| 送礼达人 | 1000 |
| 群主真爱粉 | 2000 |
| 话唠之王 | 5000 |

## 数据库结构

### achievements 表

| 字段 | 类型 | 描述 |
|------|------|------|
| id | string | 成就ID |
| name | string | 成就名称 |
| description | string | 成就描述 |
| icon | string | 成就图标 |
| points | int | 奖励积分 |
| type | string | 成就类型 |
| condition | string | 解锁条件 |
| created_at | datetime | 创建时间 |

### user_achievements 表

| 字段 | 类型 | 描述 |
|------|------|------|
| user_id | string | 用户ID |
| achievement_id | string | 成就ID |
| unlocked_at | datetime | 解锁时间 |
| progress | int | 完成进度 |
| is_completed | bool | 是否完成 |

## 使用示例

### 查看所有成就
```
/成就 列表
```

### 查看我的成就
```
/成就 我的
```

### 查看成就进度
```
/成就 进度
```

### 查看成就排行
```
/成就 排行
```

## 注意事项

- 成就系统需要数据库支持
- 成就解锁条件会根据用户行为自动检测
- 成就奖励会自动发放到用户账户
- 成就排行榜每小时更新一次
