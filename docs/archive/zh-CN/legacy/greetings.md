# Greetings Plugin Documentation

## Overview
The Greetings plugin provides greeting features for the bot, including:
- 早安 (Good Morning)
- 晚安 (Good Night)
- 欢迎语 (Welcome Messages)

## Commands

### !早安
Send a good morning greeting.

**Examples:**
```
!早安
```

### !晚安
Send a good night greeting.

**Examples:**
```
!晚安
```

### !欢迎 <user>
Send a welcome message to a user.

**Parameters:**
- `<user>`: The user to welcome

**Examples:**
```
!欢迎 张三
!欢迎 李四
```

## Configuration

The Greetings plugin does not require any additional configuration.

## Notes
- Greeting messages are pre-defined and cannot be customized
- Welcome messages can be used to greet new members in a group
- The bot will respond with appropriate emojis for each greeting
