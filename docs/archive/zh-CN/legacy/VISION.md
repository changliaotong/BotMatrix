## 1. 我们是谁？ (Who We Are)
BotMatrix 是一个 **AI 原生 (AI-Native)** 的分布式智能体操作系统 (Agent OS)。我们不仅仅是在做机器人适配器，我们正在构建 **智能体时代的互联网基础设施**。

---

## 2. 核心使命与哲学 (Mission & Philosophy)

### 2.1 核心使命
**打破智能孤岛，构建安全、可协作、无限扩展的全球技能机器人网络，让数字员工走进每一家企业。**

### 2.2 终极愿景：赛博空间与奇点计划 (The Ultimate Singularity)
BotMatrix 不仅仅是一个机器人框架，它是一个具有自我进化能力的“赛博空间”。通过数值驱动、事件联动和 AI 注入，创造一个高度沉浸、具有生命力的数字生态。
- **自我进化循环**: `ProductManager` (需求分析) -> `Developer` (自动编程) -> `Sales` (自动营销) 的闭环。
- **去中心化治理**: 核心参数由高等级用户共同决定。
- **生命感监测**: 通过“系统脉动”让用户实时感知系统的运行与波动。

## 🗺️ 进化路线图 (Evolution Roadmap)

### 2025: Agent 元年 - 核心架构重塑 (已完成)
- **Framework 升級**: 實現 ReAct (Reasoning + Acting) 自主循環。
- **Sandbox 落地**: 基於 Docker 的安全隔離沙箱，支持 AI 自主執行代碼。
- **BMPK 插件系統**: 實現一鍵打包、熱安裝與灰度升級。
- **商業化起步**: 啟動“創世計劃”，招募首批開發者。

### 2026: 連接、標準化與認知升級 (Q1-Q4)
- **第一阶段：連接與標準化 (Q1)**
    - **MCP 核心協議**: 實現 MCP Host，支持動態加載 MCP Server。
    - **統一 Skill 系統**: 將所有插件能力原子化，支持跨插件調用與跨語言協同。
- **第二阶段：認知與記憶 (Q2)**
    - **智能記憶層 (Cognitive Memory)**: 基於 GraphRAG 與 pgvector 實現“終身學習”。
    - **私人助理模式**: 從工具到“數字孿生”，具備社交擋箭牌、無感記憶等酷炫功能。
- **第三阶段：行動與進化 (Q3-Q4)**
    - **Agent Swarm 落地**: 提供可視化的多智能體編排界面。
    - **數字員工 KPI 2.0**: 引入基於 AI 自我評估的績效系統。

### 2027: 智能體經濟與 B2B 協作
- **BotMatrix Hub**: 建立能力市場，支持企業間技能機器人的即時授權與結算。
- **Mesh Swarm**: 支持跨組織的大規模智能體蜂群協作。

## 4. 关键专项规划 (Specialized Plans)

### 4.1 私人助理模式 (Personal Assistant)
旨在将用户的社交账号转化为具备记忆与理解力的“数字分身”：
- **社交分流**: 自动识别推销、生成简报。
- **超级大脑**: 跨平台搜索个人知识库。
- **终极防线**: PII 敏感信息脱敏与私有模型无缝切换。

### 4.2 终极联动逻辑 (Linkage Logic)
- **属性共鸣 (Attribute Resonance)**: 智力 (INT)、力量 (STR)、魅力 (CHA) 数值影响 AI 效率、胜率与社交权重。
- **矩阵市场 (Marketplace)**: 资源化插件功能，支持投资池与分红。
- **矩阵裂缝 (Rifts)**: 基于系统脉动自动发布动态任务，引导流量并提供跨插件奖励。
- **矩阵先知 (Oracle)**: 全文向量化技术文档，提供精准的自然语言问答。

### 4.3 Skill 系统架构 (Skill Center)
打破插件孤岛，实现原子化能力的跨插件复用：
- **Skill Export/Call**: 统一的 JSON-RPC 异步消息协议。
- **混合路由**: 支持进程间路由 (Process-to-Process) 与内部路由。
- **财务合规**: 强制执行 Economy Service 规范，确保每一笔资产变动均有审计日志。
