# Lottery 插件文档

## 功能描述
Lottery插件是一个抽签插件，支持抽签和解签功能，包含五种不同等级的签文。

## 命令格式
- `!lottery`: 进行抽签
- `!抽签`: 进行抽签
- `!interpret`: 进行解签
- `!解签`: 进行解签

## 示例
```
用户: !抽签
机器人: 🎐 抽签结果 🎐
签名：上上签
签文：久旱逢甘雨，他乡遇故知。洞房花烛夜，金榜题名时。
解签：此签为上上大吉，诸事顺遂，心想事成。
```

```
用户: !解签
机器人: 📜 解签结果 📜
签名：上签
签文：春风得意马蹄疾，一日看尽长安花。
解签：此签为上吉，事业有成，前程似锦。
```

## 功能特点
- **每日抽签限制**：每个用户每天只能抽签一次
- **五种签文等级**：上上签、上签、中签、下签、下下签
- **自动解签**：抽签时自动显示解签内容
- **独立解签**：支持单独解签功能
- **随机抽签**：使用随机数生成器确保公平性

## 签文说明
1. **上上签**：诸事顺遂，心想事成
2. **上签**：事业有成，前程似锦
3. **中签**：遇事需耐心等待，转机将至
4. **下签**：诸事不顺，需谨慎行事
5. **下下签**：遇事需格外小心，避免冲动
