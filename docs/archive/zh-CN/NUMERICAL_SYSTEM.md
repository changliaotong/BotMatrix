# 💎 BotMatrix 数值系统设计白皮书 (The Singularity Plan)

## 1. 核心公式 (Core Formula)

为了平衡早期成就感与长期挑战性，采用分段多项式公式：

`ExpForLevel(L) = 50 * L^2 + 150 * L - 200`

## 2. 经验获取 (EXP Sources)

通过 **EventNexus** 动态加权计算：

| 行为类型 | 事件来源 | 经验计算方式 | 备注 |
| :--- | :--- | :--- | :--- |
| **基础收益** | `PointTransactionEvent` | `Amount * 0.8` | 获得积分 |
| **消费奖励** | `PointTransactionEvent` | `Math.Abs(Amount) * 1.2` | 消耗积分 (支出) |
| **社交活跃** | `MessageReceivedEvent` | `10 EXP / Message` | 每日上限 500 |
| **里程碑** | `SystemInteractionEvent` | `Fixed Bonus` | 首次互动、特定节日 |

## 4. 经济平衡 (Economic Equilibrium)

- **通胀控制**：通过高等级的“积分回收”机制（如高阶勋章购买、位面晋升仪式）来回收市场过剩积分。
- **活跃激励**：等级越高，其对系统的贡献（如作为节点参与运算或管理）获得的积分报酬越高，形成正向循环。
