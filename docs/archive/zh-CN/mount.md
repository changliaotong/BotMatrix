# 🏇 超级坐骑系统 (Super Mount System)

超级坐骑系统是一个为机器人深度定制的游戏化插件，旨在为用户提供坐骑捕捉、培养、进化及属性加成的丰富玩法。

## 🌟 核心功能

- **野外捕捉**：通过随机事件在野外发现并驯服各种稀有度的坐骑。
- **属性系统**：包含移动速度、力量、幸运值和耐力四大核心数值。
- **骑乘加成**：召唤坐骑后，用户将获得基于坐骑属性的全方位能力提升。
- **等级成长**：通过持续训练提升坐骑等级，解锁更强大的属性潜力。
- **ASCII 艺术展示**：每个坐骑都有独特的视觉展示效果。

## 🎮 指令说明

| 指令 | 别名 | 说明 |
| :--- | :--- | :--- |
| `寻找坐骑` | `capture` | 消耗精力在野外寻找并尝试驯服新坐骑。 |
| `我的坐骑` | `mounts` | 查看当前拥有的所有坐骑及其详细状态。 |
| `召唤坐骑 [名字]` | `ride` | 召唤指定的坐骑进入“骑乘中”状态，获得属性加成。 |
| `坐骑训练` | `train` | 对当前骑乘的坐骑进行训练，获取经验值。 |
| `坐骑休息` | `rest` | 让疲惫的坐骑进入休息状态，恢复耐力。 |

## 💎 稀有度阶梯

坐骑分为五个稀有度等级，稀有度越高，基础属性和成长上限越高：

1.  **⚪ 普通 (Common)**：随处可见，但胜在忠诚稳健。
2.  **🔵 优秀 (Rare)**：具备一定的灵性，属性均衡。
3.  **🟣 史诗 (Epic)**：罕见的异兽，拥有出色的成长潜力。
4.  **🟠 传说 (Legendary)**：存在于神话中的生灵，拥有极高的初始属性。
5.  **🔴 神话 (Mythic)**：至高无上的存在，足以撕裂时空的强者。

## 🛠️ 开发者说明

### 数据模型
坐骑数据存储在 `UserMounts` 表中，主要逻辑位于：
- `MountModels.cs`: 定义实体类与模板。
- `MountService.cs`: 处理业务逻辑与指令交互。

### 扩展坐骑
在 `MountModels.cs` 的 `MountTemplate.All` 字典中添加新的模板即可快速扩展坐骑种类。
